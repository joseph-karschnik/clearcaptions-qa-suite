# ClearCaptions QA Testing Suite - Presentation

## Overview

This is a professional, modern web presentation showcasing the ClearCaptions QA Testing Suite. It's designed to be presented to stakeholders, management, or during interviews to demonstrate the comprehensive testing framework.

## Files

- `index.html` - Main presentation page
- `styles.css` - Modern, professional styling
- `script.js` - Interactive animations and effects

## Opening the Presentation

### Option 1: Direct File Opening
Simply open `index.html` in any modern web browser:
- Double-click `index.html`
- Or right-click → Open with → Browser

### Option 2: Local Server (Recommended)
For best experience, serve via a local server:

```bash
# Python 3
cd presentation
python -m http.server 8000

# Then open: http://localhost:8000
```

```bash
# Node.js (if you have http-server installed)
npx http-server presentation -p 8000
```
